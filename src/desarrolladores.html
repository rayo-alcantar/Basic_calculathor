Documentación Técnica para Desarrolladores

Esta guía está destinada a quienes deseen contribuir al desarrollo de la Calculadora Básica, entender su arquitectura o añadir nuevas funcionalidades.

Obtener el Código Fuente

El proyecto está alojado en GitHub. Puedes clonar o descargar el repositorio desde aquí: rayo-alcantar/Basic_calculathor

Estructura del Proyecto

La aplicación está desarrollada en Python con la biblioteca gráfica wxPython. Los archivos clave son:

  • src/basic_calculathor.py: Contiene la lógica de la interfaz de usuario (UI), la gestión de eventos y la interacción con el usuario.
  • src/operaciones.py: Alberga las clases y funciones que realizan todos los cálculos matemáticos y lógicos.
  • src/calculadora_basica.iss: Script de Inno Setup para crear el instalador de Windows.
  • update.py: Script para gestionar las actualizaciones automáticas de la aplicación.

Arquitectura

La aplicación sigue un diseño modular que separa la interfaz de usuario de la lógica de negocio:

  • Capa de Presentación (basic_calculathor.py): Responsable de todos los elementos visuales, ventanas de diálogo y eventos de usuario.
  • Capa de Lógica de Negocio (operaciones.py): Contiene la implementación pura de las operaciones, sin depender de la interfaz gráfica.

Esta separación facilita el mantenimiento, las pruebas y la escalabilidad del proyecto.

Cómo Añadir Nuevas Operaciones

Para extender la funcionalidad de la calculadora, sigue estos pasos:

  1. Implementa la lógica: En operaciones.py, añade una nueva función o una clase que contenga el algoritmo de tu nueva operación. Asegúrate de que esté bien documentada y probada.
  2. Crea la interfaz de usuario: En basic_calculathor.py, diseña una nueva ventana de diálogo (wx.Dialog) para la operación. Sigue el estilo de las existentes, como DialogoAritmetica.
    # Ejemplo de una nueva clase de diálogo class MiNuevoDialogo(wx.Dialog): def __init__(self, parent): super(MiNuevoDialogo, self).__init__(parent, title="Mi Nueva Operación") # ... aquí va el código para añadir campos de entrada y botones ...
  3. Integra la operación:
    • Añade la nueva operación al menú principal en basic_calculathor.py para que los usuarios puedan seleccionarla.
    • En el evento del botón "Calcular" de tu nuevo diálogo, llama a la función correspondiente que creaste en operaciones.py y muestra el resultado.
